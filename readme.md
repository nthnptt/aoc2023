## Usage
### Run all
```sh
php src/main.php aoc:all
```
### Run specific day
- With my data
```sh
php src/main.php aoc:day 1
```
- With custom data
```sh
php src/main.php aoc:day 1 -f my_file.txt
```
- Run last day
```sh
php src/main.php aoc:day
```
